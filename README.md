# PpPrimez

Prints the multiplication tables for n primes.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pp_primez'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pp_primez

## Usage

```
bundle exec pp_primez --count 10
```

For the test suite:

```
rake test
```
